# BCDA 题库看题小程序

纯 HTML 单文件题库浏览器，双击即可使用，零依赖。

## 功能

- **题型分类**：单选（492题）、多选（204题）、判断（222题），独立编号和进度条
- **看题模式**：直接浏览题目和答案
- **答题模式**：先作答再对答案，自动跳转下一题
- **收藏 & 错题本**：标记重点题和错题，支持筛选
- **题目解析**：可折叠的解析面板
- **键盘快捷键**：←/→ 翻页、空格显答案、P 解析、1-4 选项、F 搜索、R 随机
- **搜索功能**：关键词实时过滤，命中高亮
- **深色/浅色主题**：一键切换
- **学习记录**：自动保存进度，下次打开提示续学
- **跳题功能**：输入题号直接跳转
- **移动端适配**：响应式布局

## 使用方法

1. 下载 `quiz-viewer.html`
2. 双击用浏览器打开
3. 开始学习

所有学习数据（收藏、错题、进度）保存在浏览器 localStorage 中。

## 源数据

`BCDA题库（250418更新）.xls` — 918 道题目，GBK 编码。

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| ← / A | 上一题 |
| → / D | 下一题 |
| 空格 | 显示/隐藏答案 |
| P | 展开/折叠解析 |
| 1-4 | 答题模式选择 A/B/C/D |
| Y / N | 判断正误 |
| S | 收藏/取消收藏 |
| F | 搜索 |
| R | 随机跳题 |
| ? | 快捷键面板 |
